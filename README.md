LibVisualSLAM
=============
LibVisualSLAM is an algorithm library accompany with [CoSLAM](https://github.com/danping/CoSLAM).

System requirements
-------------

* Linux Unbuntu or Linux Mint (64 bit). 


Download & Installation
-------------

    git clone https://github.com/danping/LibVisualSLAM
    cd LibVisualSLAM
    cmake .
    make
    sudo make install
